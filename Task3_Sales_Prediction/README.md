**# Sales Prediction using Python

## Objective
The goal of this project is to predict product sales based on advertising expenditure.

## Dataset
The dataset contains advertising budgets for TV, Radio, and Newspaper along with sales.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Steps
1. Load dataset
2. Preprocess data
3. Train Linear Regression Model
4. Predict Sales
5. Visualize results using graphs

## Conclusion
The model predicts sales based on advertising budget and shows the relationship between advertising investment and product sales.# Sales Prediction using Python
